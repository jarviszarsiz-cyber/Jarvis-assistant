# Как получить APK через GitHub (Бесплатно и без компьютера)

Если у вас нет мощного компьютера или настроенной среды разработки, вы можете использовать GitHub для сборки приложения.

## Шаг 1: Создайте аккаунт на GitHub
Если у вас его нет, зарегистрируйтесь на [github.com](https://github.com). Это бесплатно.

## Шаг 2: Создайте новый репозиторий
1. Нажмите на иконку **"+"** в верхнем правом углу и выберите **"New repository"**.
2. Назовите его, например, `jarvis-assistant`.
3. Сделайте его **Public** или **Private** (на ваш выбор).
4. Нажмите **"Create repository"**.

## Шаг 3: Загрузите файлы
Вы можете загрузить файлы прямо через браузер:
1. В созданном репозитории нажмите **"uploading an existing file"**.
2. Перетащите все файлы из папки `jarvis-assistant`, которую я вам дал (включая папку `.github`).
3. Нажмите **"Commit changes"**.

## Шаг 4: Запуск сборки
1. Перейдите на вкладку **"Actions"** в вашем репозитории.
2. Слева выберите ворклоу **"Build Android APK"**.
3. Нажмите кнопку **"Run workflow"** (если она есть) или просто подождите — сборка начнется автоматически после загрузки файлов.
4. Дождитесь завершения (появится зеленая галочка, это займет около 5-10 минут).

## Шаг 5: Скачайте APK
1. Нажмите на завершенную сборку в списке.
2. Прокрутите вниз до раздела **"Artifacts"**.
3. Там вы увидите файл **`jarvis-assistant-apk`**. Нажмите на него, чтобы скачать.
4. Вы получите ZIP-архив, внутри которого будет ваш APK-файл.

## Шаг 6: Установка на телефон
1. Перекиньте скачанный файл на телефон.
2. При установке телефон может предупредить: *"Заблокировано Play Защитой"* (так как приложение не из магазина).
3. Нажмите **"Все равно установить"**.
4. Дайте приложению все разрешения (Микрофон, Работа в фоне), когда оно запустится.

**Готово! Теперь ваш Джарвис живет в вашем телефоне.**
